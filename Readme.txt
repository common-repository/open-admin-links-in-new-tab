This plugin enhances your WordPress experience by opening admin menu links and "Add New" button links for default and custom post types in a new tab.

== Description ==
* Easily open admin menu links in a new tab when enabled in the settings.
* Conveniently open "Add New" button links for selected default and custom post types in a new tab when enabled in the settings.
* Enjoy a customizable settings page to enable or disable these features and select which post types to apply them to.

== Installation ==
1. Upload the `open-admin-link-in-new-tab` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Go to the plugin settings page under \"Settings\" > \"Open Admin Link in New Tab\" to configure options.

== Frequently Asked Questions ==
= How do I enable opening admin menu links in new tabs? =

You can enable this feature by navigating to the plugin settings page under \"Settings\" > \"Open Admin Link in New Tab\" and checking the corresponding checkbox.

= Can I choose which post types\' \"Add New\" button links open in new tabs? =

Yes, you can select the post types for which you want to open \"Add New\" button links in new tabs from the plugin settings page.
